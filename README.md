# 图书管理系统 python150

## 项目概述

本图书管理系统是基于 Python 语言、Django 框架以及 SQLite 数据库构建的，旨在为图书馆提供一套功能全面、易于管理的图书管理系统。

## 技术栈

- **后端**: Python 3.x, Django
- **数据库**: SQLite
- **前端**: HTML, CSS, JavaScript

## 功能模块

### 登陆功能
- 用户认证
- 权限管理

### 图书管理
- 图书信息录入
- 图书信息查询
- 图书分类管理

### 借阅管理
- 借阅记录
- 还书提醒
- 超期处理

### 个人管理
- 个人信息维护
- 借阅历史查询

### 会员管理
- 会员信息管理
- 会员借阅权限设置

### 数据可视化图表
- 借阅数据分析
- 图书流通统计

## 系统角色

- 系统管理员
- 图书管理员
- 读者用户

## 运行环境

- Python 3.x
- Django 2.x
- SQLite 3.x
- 支持的现代Web浏览器

## 部署步骤

1. 安装 Python 3.x
2. 安装 Django
3. 创建 SQLite 数据库
4. 运行 Django 项目
5. 根据需求配置环境

## 核心亮点

- 使用 Django 的 ORM 系统，方便进行数据库操作。
- 前后端分离，易于维护和扩展。
- 数据可视化，直观展示图书借阅情况。


- 本系统为虚拟产品，不提供退换服务。
- 源代码和配置文档已提供，方便用户自行部署和二次开发。
